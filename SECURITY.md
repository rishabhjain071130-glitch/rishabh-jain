# Security Policy

## Supported Versions

Only the current main branch of the portfolio receives active security updates and patches.

| Version | Supported |
| ---     | ---       |
| v1.0.x  | Yes       |
| < v1.0  | No        |

## Reporting a Vulnerability

If you discover a potential security vulnerability (e.g. key exposure, script sanitization leaks, rate-limiting bugs, dependency security alerts), please do not report it publicly via GitHub Issues. 

Instead, report it directly by emailing the repository owner at **rishabhjain071130@gmail.com**. We will review your email and respond with appropriate coordination steps within 48 hours.
